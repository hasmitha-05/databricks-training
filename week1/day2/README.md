# SQL Joins Practice – Day 2

This folder contains SQL JOIN practice questions and solutions using the following tables:

- students
- courses
- enrollments
- instructors

## Topics Covered

- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- CROSS JOIN
- UNION
- Handling NULL values
- Multi-table joins

## Practice Questions

1. Display all students and the courses they are enrolled in.
2. Find all courses with no students enrolled.
3. Display all instructors and the courses they teach.
4. Find courses without instructors.
5. Display students and enrollment information using RIGHT JOIN.
6. Find students not enrolled in any course.
7. Use FULL OUTER JOIN to display all students and enrollments.
8. Find courses that never appeared in enrollments.
9. Display instructors and courses using FULL OUTER JOIN.
10. Create a report showing student name, course name, and instructor name.

## Bonus Challenge

Used CROSS JOIN to display every student with every course combination.

## Learning Outcomes

Through this practice, I learned:
- How different JOINs work
- Combining multiple tables
- Using UNION to simulate FULL OUTER JOIN in MySQL
- Retrieving matched and unmatched records
- Writing optimized SQL queries
