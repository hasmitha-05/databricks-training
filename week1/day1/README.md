# Databricks SQL Training Repository

This repository contains my SQL practice work, assignments, and learning progress completed during Databricks SQL training.

I am learning SQL concepts through hands-on practice using DB Fiddle and GitHub. This repository helps me improve my database management, query writing, and problem-solving skills through daily practice.

The practice work includes database creation, table creation, filtering, joins, aggregate functions, grouping, date queries, and nested queries using Employee and Department tables.
